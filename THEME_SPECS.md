# Dracpurp: The 300 IQ Semantic Architecture

## 1. The Philosophy: Semantic Primacy
Standard themes rely on Syntax (TextMate regex) which is often fragile and context-blind. Dracpurp (Optimized) prioritizes **Semantics** (Language Server Protocol), mapping colors to memory allocation and system architecture.

### The OOP Memory Map
| Semantic Concept | Category | Hex | Psychological Role |
| :--- | :--- | :--- | :--- |
| **Logic & Contracts** | `class`, `interface`, `namespace` | #B2A1E5 | Heavy, heap-allocated engines. |
| **Data Shapes** | `type`, `struct`, `enum`, `<T>` | #A8B4C3 | Passive boilerplate. Recedes visually. |
| **Data Vehicles** | `parameter` | #DEAC77 | Track data passing *between* classes. |
| **Executables** | `method`, `function` | #A1D3A9 | Active behavior. Green = "Go". |
| **Control Flow** | `keyword` | #F272BE | The structural skeleton (`if`, `return`). |
| **The Workhorse** | `variable`, `string`, `number` | #CEC8B9 | Brightest element. Focus on actual state. |

## 2. Contrast Physics & Optical Engineering
To prevent halation and retina burn while maintaining clarity:
- **Normal Background:** #282A36. Eggshell contrast is capped at ~8.5:1.
- **HC Background:** #111115. Eggshell contrast spikes to ~11.2:1 for high-glare environments.
- **Motion:** Italic flow (15-degree slant) is applied to `keywords`, `parameters`, and `comments` in Night Owl variants to allow faster skimming of boilerplate.

## 3. The TextMate Fallback
For languages like CSS/SCSS where semantic tokens are limited, we target `support.function` and `entity.name.function` to ensure the Mint Green executable color persists.

## 4. UI Tiering
- **Editor Background:** The core canvas.
- **Sidebar Background:** Slightly darker than the editor for depth.
- **Activity/Title Bar:** The darkest "abyssal" tier for peripheral containment.
